# Installation

`@esfront/react` is available as an [npm package](https://www.npmjs.com/package/@esfront/react). To install and save
dependencies in your `package.json`:

```bash
npm install @esfront/react @mui/material @emotion/react @emotion/styled
```

Then follow the installation instructions for [MUI](https://mui.com/getting-started/installation/).
