# Localization

The default locale is English. If you want to use other locales, use the theme to configure the locale text globally:

```tsx
import { useMemo } from 'react';

import { ThemeProvider, FormatSize, ru } from '@esfront/react';
import { ruRU } from '@mui/material/locale';

const Application = () => {
  const theme = createTheme(
    {
      paletteDark: {
        ...palettes.common,
        ...palettes.dark,
      },
      paletteLight: {
        ...palettes.common,
        ...palettes.light,
      },
    },
    { ...ruRU, ru }
  );

  return (
    <ThemeProvider theme={theme}>
      <FormatSize>1024</FormatSize>
    </ThemeProvider>
  );
};
```

You can [find the source](https://github.com/Elonsoft/esfront/blob/master/packages/react/src/components/locale/index.ts)
in the GitHub repository.

To create your own translation, or to customize the English text, copy this file to your project, make any changes
needed and import the locale from there.
